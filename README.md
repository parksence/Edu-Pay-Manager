# 학원 수강료 관리 (Edu-Pay Manager)

수강료 자동 계산, 학부모/원장님용 엑셀 내보내기가 가능한 웹 앱입니다.

## 기술 스택

- React 18 + Vite + TypeScript
- Tailwind CSS, Radix UI 기반 UI (shadcn 스타일)
- Zustand (상태 관리 + localStorage persist)
- SheetJS (xlsx) 엑셀 불러오기/내보내기
- Node v24.13.1
- npm 11.8.0

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속.

## 빌드

```bash
npm run build
```

## 주요 기능

- **스마트 테이블**: 테이블에서 이름, 구분, 할인(원), 셔틀, 교재비, 결석 차감을 직접 수정하면 최종 금액이 즉시 계산됩니다.
- **이번 달 총 예상 매출액**: 상단 카드에 합계 표시.
- **엑셀 불러오기**: 기존 명단 엑셀(컬럼: 이름, 구분, 수학 수강료, 할인, 셔틀, 교재비 등)을 업로드하면 리스트로 불러옵니다. 구 양식의 형제할인(O) 열도 읽어 5% 할인액으로 반영합니다.
- **엑셀 내보내기**: 계산된 결과(기본수업료, 셔틀비, 최종금액 포함)를 엑셀 파일로 다운로드합니다.
- **데이터 유지**: 새로고침해도 localStorage에 저장된 데이터가 유지됩니다.

## 수강료 계산 공식

최종 금액 = max(0, 기본 수업료 + 수학 수강료 − 할인) + 셔틀비 + 교재비 − 결석 차감

- 구분별 기본 수업료·셔틀 요금은 `src/lib/constants.ts`에서 확인·수정할 수 있습니다.
