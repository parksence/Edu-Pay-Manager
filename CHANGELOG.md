# Edu-Pay-Manager 릴리즈 노트

## [0.1.0] - 2026-02-24

**학원 수강료 관리** – 수강료 자동 계산, 엑셀, 원장님 보고용 기능을 갖춘 웹 앱입니다.

### 새 기능

**원장님 보고용**

- **KPI 대시보드**: 이번 달 예상 매출, 학생 수(초/중/고), 기타 수익(교재비), 미수금·차감, 미납 인원/합계를 한 화면에 표시 (모바일에서 바로 확인 가능, PDF 제거)
- **카카오톡 요약 복사**: 결산일자, 매출 요약, 항목별, 미납자 명단을 카톡에 붙여넣기용 텍스트로 복사 (복사 완료 토스트 표시)
- **원장님 브리핑 모드**: 상단 토글로 활성화 시 메인 수정 테이블 숨김, 카드·차트만 표시 (모바일 가독성 강화)

**데이터 시각화**

- **구분별 매출 비중**: 초/중/고 Pie 차트 (recharts)
- **수납 vs 미납**: 가로형 누적 막대 차트, 수납(녹색) / 미납(빨간색), 막대 위 금액 라벨

**수강료·납부 관리**

- **납부 완료 / 납부 금액**: 테이블에서 행별 납부 여부 체크 및 납부 금액 입력 (미입력 시 기본값: 납부 완료)
- **비고 컬럼**: 메인 테이블·미납 명단·엑셀 내보내기/불러오기에서 비고 입력·표시
- **연락처**: 학생별 연락처 입력, 미납 명단에서 전화 걸기 버튼으로 `tel:` 링크 제공

**엑셀**

- 엑셀 양식 다운로드 / 불러오기 / 내보내기
- 내보내기 시 자동 계산 항목(기본수업료, 셔틀비, 최종금액) 컬럼 제외
- 내보내기 헤더 줄바꿈 방지(열 너비 설정)

**배포**

- GitHub Pages 배포 지원 (GitHub Actions 워크플로)

### UI/UX 개선

- **레이아웃**: 화면 전체 너비 사용 (`max-w` 제거)
- **대시보드**: Primary KPI(총 매출) Indigo 그라데이션 강조, KPI 4카드 그리드, 미납 인원 Rose 배경·Badge
- **미납자 명단**: 테이블 → 2단 카드 → 멀티 컬럼 그리드(데스크탑 2~3열), 스크롤 영역 최대 높이 600px, 스티키 헤더, 비고 한 줄 + 호버 시 Tooltip 전체 내용
- **테이블**: 이름·최종 금액 굵게, 입력 포커스 링크 Indigo, 최종 금액 헤더 툴팁(계산 공식), 헤더 줄바꿈 방지(형제 할인, 미납액, 납부 완료, 기본 수업료, 셔틀비, 최종 금액)
- **미납액**: 납부 금액이 최종보다 클 때 마이너스 금액을 빨간색으로 표시
- **헤더**: 자동 저장 안내, 수강료 산정 공식 툴팁

### 기술 스택

- React 18, Vite, TypeScript, Tailwind CSS, Zustand(persist), shadcn/ui(radix), lucide-react, recharts, xlsx
