# Edu-Pay-Manager 릴리즈 노트

## [0.4.0] - 2026-03-20

### 추가/변경

- **테이블 컬럼 설정**: 상단「컬럼 설정」에서 표시할 열만 체크박스로 토글 (드롭다운, 바깥 클릭 시 닫힘)
- **할인 입력**: 형제 할인(체크)을 제거하고 **할인(원)** 숫자 입력으로 변경. 최종 금액 = `max(0, 기본+수학 − 할인) + 셔틀 + 교재 − 결석차감`
- **엑셀**: 컬럼명 `할인`(원). 구 양식의 `형제할인`(O) 열은 불러올 때 5% 할인액으로 환산
- **구 저장 데이터**: `형제할인: true`만 있고 할인이 0인 행은 계산 시 동일하게 5% 환산
- **학부모용 멘트**: `N월` 표기를 **오늘 기준 다음 달**로 통일 (12월 → 다음 해 1월)
- **테이블 헤더**: 모든 `TableHead` 기본 정렬을 가운데로 통일
- **상세검색**: 이름·구분만 유지 (할인·셔틀 필터 제거)

### 제거

- **납부·연락처·미납 흐름**: 테이블의 납부 완료·납부 금액·연락처·미납액 열, 납부 상세검색, 전체 체크/해제, KPI 미납 명단·배지, 통계「수납 완료 vs 미납」막대 차트, 카톡 요약의 미납 블록, 모바일 KPI 공유 스냅샷의 미납 필드, 엑셀 납부·연락처 컬럼

### 문서·기타

- **README·앱 내 공식 문구**: 위 할인·최종 금액 식으로 갱신
- **항목별 매출 비중 차트**: 할인을 수업료(영어/수학) 합산에 비례 배분해 반영

## [0.3.0] - 2026-03-04

### 추가/변경

- **대시보드 차트**: 항목별 매출 비중(영어/수학/셔틀/교재비), 구분별 학생 수 비중(초/중/고) 차트 추가
- **차트 색상**: 구분별 매출 비중·학생 수 비중에서 초등(파랑)·중등(주황)·고등(초록)으로 구분되도록 색상 조정

## [0.2.0] - 2026-02-24

### 추가/변경

- **학부모용 멘트**: 테이블에 학부모용 멘트 복사 버튼 추가 (헤더 "학부모용 멘트", 버튼 라벨 "멘트 복사"). 복사 시 토스트로 안내
- **학부모용 멘트 문구**: 0원인 항목(수강료·셔틀비 등)은 멘트에서 제외
- **테이블 작업 열**: 멘트 복사와 삭제 버튼을 열 분리 — "학부모용 멘트" / "작업" 헤더로 구분
- **수학 수강료**: 구분 컬럼 다음에 수학 수강료 입력란 추가. 입력한 금액은 기본 수업료에 포함되어 최종 금액에 반영 (할인 적용 후 + 셔틀 + 교재비 - 결석 차감)
- **교재비 사유**: 교재비 항목 다음에 교재비 사유 입력란 추가
- **행 추가**: 행 추가 시 목록 맨 위에 새 행이 추가되도록 변경
- **기본 정렬**: 테이블 기본 정렬을 이름 가나다순(오름차순)으로 적용
- **엑셀**: 내보내기/불러오기/양식에 수학 수강료, 교재비 사유 컬럼 반영

## [0.1.0] - 2026-02-24

**학원 수강료 관리** – 수강료 자동 계산, 엑셀, 원장님 보고용 기능을 갖춘 웹 앱입니다.

### 새 기능

**원장님 보고용**

- **KPI 대시보드**: 이번 달 예상 매출, 학생 수(초/중/고), 기타 수익(교재비), 미수금·차감, 미납 인원/합계를 한 화면에 표시 (모바일에서 바로 확인 가능, PDF 제거)
- **카카오톡 요약 복사**: 결산일자, 매출 요약, 항목별, 미납자 명단을 카톡에 붙여넣기용 텍스트로 복사 (복사 완료 토스트 표시)
- **원장님 브리핑 모드**: 상단 토글로 활성화 시 메인 수정 테이블 숨김, 카드·차트만 표시 (모바일 가독성 강화)

**데이터 시각화**

- **구분별 매출 비중**: 초/중/고 Pie 차트 (recharts)
- **수납 vs 미납**: 가로형 누적 막대 차트, 수납(녹색) / 미납(빨간색), 막대 위 금액 라벨

**수강료·납부 관리**

- **납부 완료 / 납부 금액**: 테이블에서 행별 납부 여부 체크 및 납부 금액 입력 (미입력 시 기본값: 납부 완료)
- **비고 컬럼**: 메인 테이블·미납 명단·엑셀 내보내기/불러오기에서 비고 입력·표시
- **연락처**: 학생별 연락처 입력, 미납 명단에서 전화 걸기 버튼으로 `tel:` 링크 제공

**엑셀**

- 엑셀 양식 다운로드 / 불러오기 / 내보내기
- 내보내기 시 자동 계산 항목(기본수업료, 셔틀비, 최종금액) 컬럼 제외
- 내보내기 헤더 줄바꿈 방지(열 너비 설정)

**배포**

- GitHub Pages 배포 지원 (GitHub Actions 워크플로)

### UI/UX 개선

- **레이아웃**: 화면 전체 너비 사용 (`max-w` 제거)
- **대시보드**: Primary KPI(총 매출) Indigo 그라데이션 강조, KPI 4카드 그리드, 미납 인원 Rose 배경·Badge
- **미납자 명단**: 테이블 → 2단 카드 → 멀티 컬럼 그리드(데스크탑 2~3열), 스크롤 영역 최대 높이 600px, 스티키 헤더, 비고 한 줄 + 호버 시 Tooltip 전체 내용
- **테이블**: 이름·최종 금액 굵게, 입력 포커스 링크 Indigo, 최종 금액 헤더 툴팁(계산 공식), 헤더 줄바꿈 방지(형제 할인, 미납액, 납부 완료, 기본 수업료, 셔틀비, 최종 금액)
- **미납액**: 납부 금액이 최종보다 클 때 마이너스 금액을 빨간색으로 표시
- **헤더**: 자동 저장 안내, 수강료 산정 공식 툴팁

### 기술 스택

- React 18, Vite, TypeScript, Tailwind CSS, Zustand(persist), shadcn/ui(radix), lucide-react, recharts, xlsx
